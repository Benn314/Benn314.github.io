### Common Instruction

- `hexo clean` 清除 `public` 静态目录
- `hexo s` 本地启动默认指定 `4000` 端口预览
- `hexo g` 生成 `public` 静态目录
- `hexo d` 部署
- 
- `hexo new post “文章名”` 新建文章
- `hexo new page “菜单页面名”` 新建菜单页
